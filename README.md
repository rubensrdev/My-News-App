# My News App - Swift UI, Core Data and API

My News es una aplicación para iOS que te mantiene informado sobre noticias internacionales actuales. Con dos secciones principales, permite explorar un listado de noticias y marcar las que desees leer más tarde. Las noticias guardadas se encuentran en una sección específica, donde puedes acceder a una vista detallada de cada artículo. Desarrollada con Swift, SwiftUI y Core Data, My News ofrece una experiencia de usuario moderna y eficiente.
